Keyboard Shortcuts

Tile Grid

Add Tile: Left Click

Add Effect/Object: Left Click

Add Object Popup: T

Add Random Decoration*: Shift + Left Click

Remove Tile: Right Click

Remove Decoration: Shift + Right Click

Remove Effect/Object: Alt + Right Click

Switch to Previous Tile: P

*You'll need to add decorations via the "Random Decorations Collections" task for this to take effect

Level Editor

Zoom In: I

Zoom Out: O

Go to Top Left Corner: L

Go to Bottom Right Corner: .

Play Animations: Spacebar

Play Sounds: Shift+Spacebar

Show Only Tile Grid: F

General

Editor Selection: Esc

Help: H

Settings: S

Resources: R

Access Forums: Q

System

Exit: Crtl/Cmd + Q
